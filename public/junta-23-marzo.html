COMO USAR UTMs EN REDES

Cada link que se comparta en redes debe llevar parametros UTM para rastrear de donde vienen los leads:

  • utm_source = plataforma (instagram, facebook, tiktok, google)
  • utm_medium = tipo de contenido (story, post, reel, ad, bio)
  • utm_campaign = campaña especifica (inscripciones-2026, open-house, promo-marzo)
EJEMPLOS DE LINKS

// Instagram Bio:

carrera.arte7.net?utm_source=instagram&utm_medium=bio&utm_campaign=inscripciones-2026

// Facebook Ad:

carrera.arte7.net?utm_source=facebook&utm_medium=ad&utm_campaign=promo-marzo

// TikTok Bio:

carrera.arte7.net?utm_source=tiktok&utm_medium=bio&utm_campaign=inscripciones-2026

IMPORTANTE
  • El dashboard ya captura estos datos automaticamente
  • Se puede filtrar por fuente (utm_source) en el dashboard
  • Con esto podemos medir que canal trae mas leads y de mejor calidad (score)
  • Sin UTMs no sabemos de donde vienen los leads
